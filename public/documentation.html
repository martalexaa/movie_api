myFlix API

This web application will provide users with access to information about different movies, directors, and genres.

Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL Request Parameter HTTP Method Response body data format Request body data format
Returns a list of all movies to the user "/movies" none GET
Example: app.get("/movies", (req,res) =>{})		 A JSON object holding data about all the movies
Example: {Title: "Moulin Rouge", Description: "", Director: { Name: "Baz Luhrmann", Bio: "" }, Genre: { Name: "Musical", Description:""} }
Returns data about a single movie by title to the user "/movies/:Title" :Title GET
Example: app.get("/movies/:Title", (req,res) =>{})		 A JSON object holding data about a single movie
Example: {Title: "Title1", Description: "...."}
Returns data about a genre by name "/movies/genre/:genreName" :genreName GET
Example: app.get("/movies/genres/:genreName", (req,res) =>{})		 A JSON object holding data about a genre
Example: Genre: { Name: "Action", Description:"..."}
Returns data about a director by name "/movies/directors/:directorName" :directorName GET
Example: app.get("/movies/directors/:directorName", (req,res) =>{})		 A JSON object holding data about a director
Example: Director: { Name: "Lucas", Bio: "..." },
Allows new users to register "/users" none POST
Example: app.post("/users", (req,res) =>{})		 A JSON object holding data about new user that was added
Example: { ID: Integer, Username: String, Password: String, Email: String, Birthday: Date }, 		A JSON object is requested
Example: { ID: Integer, Username: String, Password: String, Email: String, Birthday: Date },
Allows users to update their user info "/users/:id" :id PUT
Example: app.put("/users/:id", (req,res) =>{})		 A JSON object holding the updated user data
Example: { ID: Integer, Username: String, Password: String, Email: String, Birthday: Date }, 		A JSON object is requested
Example: { ID: Integer, Username: String, Password: String, Email: String, Birthday: Date },
Allows users to add a movie to their list of favorites "/users/:id/:movieTitle" :id,:movieTitle POST
Example: app.post("/users/:id/:movieTitle", (req,res) =>{})		 A text message indicating that the movie has been added
Allows users to remove a movie from their list of favorites "/users/:id/:movieTitle" :id,:movieTitle DELETE
Example: app.delete("/users/:id/:movieTitle", (req,res) =>{})		 A text message indicating that the movie has been deleted
Allows existing users to deregister "/users/:id" :id DELETE
Example: app.delete("/users/:id", (req,res) =>{})		 A text message indicating that the user has been deleted
